🎮 How to Use¶

  1. Browse Elements

    • The periodic table displays all 118 elements in their correct positions

    • Elements are color-coded by category (alkali metals, halogens, noble gases, etc.)

  2. View Element Details

    • Click any element to open the modal

    • See basic information like atomic mass, period, and group

  3. Explore 3D Models

    • Switch to the “Bohr Model 3D” tab

    • Drag to rotate, scroll to zoom

    • View in AR on compatible devices

  4. View Other Visualizations

    • Element Image Tab - See what the element looks like physically

    • Spectral Bands Tab - View the spectral analysis

  5. Read Information

    • View comprehensive element properties

    • Read Wikipedia summaries

    • Click to view Wikipedia source

  6. Close Modal

    • Press Escape, click X, or click outside the modal

🐛 Troubleshooting¶

Issue: Periodic table didn’t load Solution: Make sure you’re using http://localhost:8000 (not file://)

Issue: 3D models not showing Solution: Check your internet connection (models load from Google CDN)

Issue: Port 8000 already in use Solution: Kill the existing process or modify the port in server.py

Issue: Page looks broken Solution: Hard refresh your browser (Ctrl+Shift+R or Cmd+Shift+R)
